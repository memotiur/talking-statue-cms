## Installation
- Customize .env file (Change db name, username and password)
- Update Composer, require command from the Terminal:
``` composer update ```
- Create all table: 
``` php artisan migrate ```
- Seed: 
``` php artisan db:seed ``` 
- Run project: 
``` php artisan serve ``` 
- Login from browser using username: admin@gmail.com, password: password
